🎨 RGB Color Picker

Una aplicación web sencilla e interactiva que permite a los usuarios personalizar el color de fondo de la pantalla en tiempo real manipulando los valores RGB (Rojo, Verde y Azul).

🚀 Características

Cambio en tiempo real: El fondo de la página se actualiza instantáneamente al mover los deslizadores.
Control Preciso: Utiliza tres controles deslizantes (input type="range") para definir cada canal de color (0-255).
Visualización de Código: Muestra el código RGB generado para que puedas copiarlo y usarlo en tus propios proyectos.
Diseño Responsivo: Interfaz minimalista que se adapta a cualquier tamaño de pantalla.

🛠️ Tecnologías utilizadas

HTML5: Estructura semántica.
CSS3: Estilos modernos y disposición con Flexbox/Grid.
JavaScript: Manipulación del DOM para actualizar el color dinámicamente.
